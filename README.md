# Jot

A simple note taking app

## Start Contributing

Install Required Dependencies (Mac Instructions)

```bash
brew upgrade

## Node
brew install node 

## Package Manager (Pick your preferred option)
### Option 1
brew install pnpm
### Option 2
npm i -G pnpm

## Supabase CLI
brew install supabase/tap/supabase

## Editor
brew install cask visual-studio-code

```
